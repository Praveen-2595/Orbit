## What Is ORBIT?

Most productivity apps ask: *"Did you do the thing?"*

ORBIT asks: *"Do you remember why the thing matters?"*

ORBIT is not a to-do list. Not a habit tracker. Not Notion.

It's a **living companion** тАФ an AI that knows your long-term dreams, your weekly goals, and your daily checklist, and connects all three into one coherent picture. The AI talks to you like a person: sometimes about your goals, sometimes about nothing in particular. It remembers what matters to you.

> *"ORBIT should make users feel like their life is moving тАФ not like they're failing a productivity system."*

## Features

### ЁЯОп Three-Layer Life System
Everything in ORBIT connects upward:
```
Daily habits & checklist
        тЖУ feeds into
Short-term goals (weeks/months)
        тЖУ feeds into  
Life Vision (1yr / 3yr / 5yr)
        тЖУ all seen by
AI Companion тАФ talks freely across all layers
```

### тЪая╕П Doom Meter
A real-time pressure gauge that rises every hour you're not progressing toward your goals. Not just a completion bar тАФ it measures your **actual pace vs required pace** with deadline-aware urgency multipliers.

```
0тАУ25%   ЁЯЯв SAFE      Calm companion. Goals are background context.
25тАУ50%  ЁЯЯб RISING    Firm. Flags what needs attention today.
50тАУ70%  ЁЯЯа WARNING   Urgent. Names what's at risk.
70тАУ85%  ЁЯФ┤ DANGER    Honest intervention. References your deadlines.
85тАУ100% ЁЯФ┤ CRITICAL  Brutally honest. Activates stakes engine.
```

### ЁЯза Stakes Engine
The feature that makes ORBIT unforgettable.

During onboarding, ORBIT asks what you're **actually risking** тАФ not your productivity goals, but the real consequences. Who depends on you. What failure looks like in your life. What success unlocks.

When doom rises above 50%, the AI doesn't say *"you're behind on DSA."*

It says *"the interview your parents are depending on is in 3 days. You've studied 2 hours."*

### ЁЯдЦ AI Companion (Claude-powered)
- Talks freely тАФ not every message is about goals
- Remembers facts from every conversation automatically
- Personality shifts with doom level
- Dynamic first message based on time of day
- Stakes-aware responses at high doom
- Free conversation mode тАФ sometimes you just need to talk

### тЬЙя╕П Letter to Future Self
Write a letter to yourself 90 days from now. Sealed. Locked. Revealed automatically on day 90.

Nobody abandons an app holding a letter from their past self.

### ЁЯОо Level System
```
тЧЛ  Drifting        тАФ finding your gravity      (0тАУ99 pts)
тЧР  Waking Up       тАФ something is shifting     (100тАУ299 pts)
тЧС  Rising          тАФ momentum is building      (300тАУ699 pts)
тЧХ  In Orbit        тАФ you found your rhythm     (700тАУ1499 pts)
тЧП  Locked In       тАФ nothing can stop this     (1500тАУ2999 pts)
тШЕ  Unstoppable     тАФ you became the goal       (3000+ pts)
```

### ЁЯНЕ Pomodoro Timer
- Circular SVG progress ring
- Short break (5m) and long break (15m) modes
- Saves progress when stopped тАФ resume or log it
- Auto-logs hours to linked goal on completion
- Custom duration settings

### ЁЯУК Overview Dashboard
- Doom trend chart (last 7 days)
- Habit completion heatmap (last 28 days)
- Focus sessions bar chart (this week)
- Goal progress bars with expected-pace markers
- Weekly summary тАФ "best day", habit rate, hours logged

### ЁЯФД Retention Hooks
- **Bad Day Protocol** тАФ when doom hits 80%+ at 8pm, AI switches to compassion not pressure
- **Miss Me Message** тАФ after 2+ days away, opens with a personal contextual message
- **Relapse Recovery** тАФ auto-adjusts goals on return, reduces doom 30%, removes shame
- **Reality Check Card** тАФ shows your own words back when you're slipping
- **Weekly Identity Report** тАФ narrative of who you were this week, not just metrics

## Getting Started

### Prerequisites
- Node.js 18+
- A Claude API key from [console.anthropic.com](https://console.anthropic.com)

### Installation

```bash
# Clone the repo
git clone https://github.com/yourusername/orbit.git
cd orbit

# Install dependencies
npm install

# Copy environment variables
cp .env.example .env.local
```

### Environment Variables

```env
# .env.local
ANTHROPIC_API_KEY=sk-ant-api03-...
NEXT_PUBLIC_APP_URL=http://localhost:3000
```

### Run Locally

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

### Deploy to Vercel

```bash
# Install Vercel CLI
npm i -g vercel

# Deploy
vercel

# Add environment variables in Vercel dashboard:
# ANTHROPIC_API_KEY = your key
```

Or just drag your folder to [vercel.com/new](https://vercel.com/new) тАФ done in 60 seconds.

|-----|----------|

| `orbit_stakes` | Life context + per-goal consequences |
| `orbit_letters` | Letters to future self |
| `orbit_points` | Gamification points + level |
| `chat_history` | Recent conversation history |
